# Exercício TypeScript - Aula 24/02

Projeto desenvolvido para praticar conceitos básicos de TypeScript.

## 📚 Conceitos aplicados

- Criação de tipos com `type`
- Arrays tipados
- Funções com tipagem definida
- Formatação de valores monetários
- Compilação de TypeScript para JavaScript

## 📦 Descrição

O projeto simula um pequeno catálogo de produtos, exibindo:

- Categoria
- Nome
- Preço formatado em Real (BRL)
- Status de disponibilidade

## 🛠 Tecnologias utilizadas

- Node.js
- TypeScript
- Git
